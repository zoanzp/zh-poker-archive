# zh-poker-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式保存，不依托于任何特定域名或网站平台。

## 项目简介

zh-poker-archive 是一个集中存放 HTML 页面的归档项目。所有页面均为独立文件，可直接在浏览器中打开查看。仓库不涉及任何动态后端或数据库，结构简洁，便于长期保存与分发。

## 目录说明

```
/
├── pages/          # 存放主要 HTML 页面
├── assets/         # 页面依赖的静态资源（图片、样式、脚本等）
└── README.md       # 本说明文件
```

- `pages/`：归档的 HTML 文件按需放置于此，文件名通常反映内容主题或编号。
- `assets/`：与页面配套使用的资源文件，保持相对路径引用。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，支持离线查看。
- 页面内容可能涉及多种主题或格式，不限定于单一领域。
- 归档目的是为了记录和保存，不承诺内容的实时更新或维护。

## 维护说明

- 本仓库不定期更新，新增页面或调整现有文件。
- 如发现页面无法正常显示或资源缺失，可提交 Issue 说明情况。
- 欢迎提交 Pull Request 贡献新的归档页面或改进现有内容，但请确保文件独立、无外部依赖。
